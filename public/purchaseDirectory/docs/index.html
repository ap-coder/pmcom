Requirements

The hosting must meet the following requirements

  1. PHP 7.3+ OR PHP 8.0+
  2. Openssl
  3. Pdo
  4. Mbstring
  5. Tokenizer
  6. JSON
  7. CURL
  8. Apache Mod_rewrite

System

  1. Backend Framework: Built on Laravel
  2. Frontend Framework: Built on Bootstrap
  3. Supports PHP 7.3+ Or 8.0+
  4. Supports MySQL, Mysqli, SQLite, pgSQL, SQLsrv

Installation

You must first have hosting on one of the companies and you have a cPanel or something similar.
You can also install the item on the local hosting.
Follow the following steps to learn how to install the item correctly

Step 1

cPanel login

  1. Navigate to your website with “/cpanel” Or “:2082” Or “:2083” added to the end of the domain name. Be sure to replace example.com with your actual domain name.

  2. You will then be on the cPanel login page. Enter your cPanel username and password, then click the Log in button.

Step 2

Create a MySQL Database in cPanel When you create a database, many users are unaware that there are several other tasks that you need to do as well. The new database creation process actually involves:

  • Creating a database
  • Creating a database user
  • Giving your database user access to work with your database

cPanel includes a MySQL Database Wizard that walks you through each of these steps.

  1. Click the MySQL Database Wizard under the Databases heading.

  2. Next to New Database enter a name for your database and click Next Step.

Warning: If you receive an error message regarding the limited number of available databases. Contact with hosting technical support

  1. Next to Username enter a username. Enter a password next to Password, enter it again for Password (Again) and then click Create User.

  2. On the next page, you’ll assign privileges for the user to the database. Check the box next to All Privileges and then click Next Step

  3. Congratulations! Now, you successfully create a database.

Step 3

upload a file using File Manager in cPanel

  1. On the home screen under the files section choose File Manager.

  2. Click the Settings icon in your File Manager window.

  3. Next, choose where you’d like to start in. For this example, we’re going to use the Web Root option – which is the public_html folder. Since we’re uploading to the public_html folder we do not need to navigate anywhere else.

  4. Select Upload from the icons at the top.

  5. Navigate to the file you want to upload on your local computer by selecting the Choose File button.

  6. Select the file and click Open. Once the file is finished uploading you can click on the link in the cetner of the screen to retrun to the folder you uploaded the file to.



  7. Right click and press Extract



  8. make sure you have those files

Step 4

Script installation

  1. Run the script installation script by accessing the URL in a web browser example.com
    Be sure to replace example.com with your actual domain name. Click Check Requirements

  2. Click Check Permissions

  3. Click Configure Environment

  4. Fill out all the data fields and Click Install

    The database name, user, and password that you made in step 2

    When this page appears, check the database name, database user name, and password for the database user.

  5. Fill out all the data fields and Click Submit

    Congratulations! Now, you successfully install Script.

Upgrade

Please make a backup of your old files.

  1. Open folder update

  2. Compress the two project , public

  3. Open the cPanel, go to the script path, and Click upload

  4. Choose update.zip file

  5. Extract the file update.zip

  6. Open admin Control and Click Update (1.5)

    If everything is fine and the update link does not appear, open the update link directly

    example.com/update

  7. Follow the update steps

Admin

open URL in a web browser example.com/admincp Be sure to replace example.com with your actual domain name.

How to change the name of admincp folder for protection

Open the file project/.env Using the editor in line 39, replace the word admincp

Do not use the space in the name. Use uppercase or lowercase letters, numbers, - , _ only
ex : new_name_admin
